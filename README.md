## Use with Google Colab

1. Open the notebook in Colab from GitHub.
2. In Colab, click `File -> Save a copy in Drive`.
3. Run the first setup cell.

The first cell in the notebook will:

- clone the repository into `/content/image-segmentation`
- move into `/content/image-segmentation`
- install the Python dependencies from `requirements.txt`
