## Use with Google Colab

1. Push this folder to GitHub.
2. Replace the placeholder GitHub URL in:
   - this README badge
   - the `REPO_URL` variable in [image-segmentation/practical_application.ipynb](/home/menir/ml_course/image-segmentation/practical_application.ipynb)
3. Open the notebook in Colab from GitHub.
4. Run the first setup cell.

The first cell in the notebook will:

- clone the repository into `/content/ml_course`
- move into `/content/ml_course/image-segmentation`
- install the Python dependencies from `requirements.txt`

## Notes for students

- Colab runtimes are temporary, so the setup cell must be rerun in each new session.
- If you want to keep your notebook edits, use `File -> Save a copy in Drive`.
